# Загрузчик картинок котов на Яндекс.Диск

Программа для загрузки милых котиков с API cataas.com на Яндекс.Диск.

## Функциональность

- Получение данных о котах с API cataas.com
- Сохранение картинок локально в папку `images/`
- Создание папок на Яндекс.Диске
- Загрузка картинок на Яндекс.Диск
- Создание JSON файла `files_info.json` с информацией о файлах
